Integration FAQ

Common questions before connecting

How long does the integration setup take?
For Shopify and WooCommerce, most connections are live within 1–2 business days once you share your store credentials. Amazon Seller Central connections typically take 2–3 days including SP-API authorization. Custom API integrations depend on your team's availability — we provide full technical documentation and our team supports you through testing.
Do you support API access for custom platforms?
Yes. Our REST API covers order creation, inventory queries, parcel status retrieval, tracking number lookup, and inbound shipment creation. Full API documentation is available to clients on request. We support authentication via API key. Our technical team answers questions during your integration build.
Can I upload orders by file if I don't want to use an API?
Yes. Our web interface accepts CSV, XML, XLS, DBF, and MDB files. You can upload manually or configure an automated FTP push on your chosen schedule. For crowdfunding campaigns, most clients upload a single CSV of backer addresses and we handle all fulfillment from there.
How do tracking numbers get back to my customers?
For direct platform integrations (Shopify, WooCommerce, Amazon), tracking numbers write back to your store automatically as soon as we dispatch the order. Your customer receives their shipping confirmation from your own store. For API and file-based setups, tracking numbers are available via API query or included in your daily status export file.
How often do parcel statuses update?
Parcel status updates are available daily as standard. For API-connected clients, you can query status in real time. We also support webhook notifications — we send a push to your endpoint when an order status changes, so you don't need to poll. The exact setup depends on your integration method.
Can you build custom reports to match our internal format?
Yes. Our standard report set covers most operational and accounting requirements. If you need a report in a specific format — to feed into your ERP, match your internal template, or combine data in a way our standard reports don't — we build that to your specification. Discuss it with your account manager during onboarding.
Is the connection secure? What about VPN access?
All API connections run over HTTPS with API key authentication. For clients with closed-network environments or compliance requirements, we support VPN-based connections for file transfer. FTP over TLS is also available. Speak to our team if you have specific security requirements and we'll confirm what we can accommodate.
